# Brainstorm de Design - Cura Coração

## Abordagem 1: Minimalismo Terapêutico com Geometria Suave
**Probabilidade:** 0.08

### Design Movement
Minimalismo contemporâneo inspirado em terapia visual e design de bem-estar.

### Core Principles
1. **Espaço Respirável**: Amplo espaçamento negativo que induz calma e introspecção
2. **Geometria Orgânica**: Formas suaves, sem ângulos agressivos, transmitindo segurança
3. **Redução Intencional**: Apenas elementos essenciais, eliminando distrações
4. **Ritmo Lento**: Transições lentas e deliberadas que convidam à pausa

### Color Philosophy
- **Paleta Principal**: Tons neutros quentes (bege, creme, cinza claro) com acentos em verde-água suave
- **Intenção Emocional**: Cores que evocam terra, natureza e repouso. Verde-água representa renovação sem agressividade
- **Contraste Suave**: Sem preto puro; usar cinza escuro para manter suavidade

### Layout Paradigm
- Estrutura vertical com seções que se revelam ao scroll
- Elementos flutuam com espaçamento assimétrico
- Cartões com bordas arredondadas extremamente suaves (radius grande)
- Sem grid rígido; fluxo orgânico de conteúdo

### Signature Elements
1. **Formas Fluidas**: Dividers com ondas suaves, sem linhas retas
2. **Ícones Minimalistas**: Linhas finas, sem preenchimento sólido
3. **Espaço em Branco Estratégico**: Grandes áreas vazias entre seções

### Interaction Philosophy
- Hover effects sutis (mudança de cor muito leve)
- Cliques produzem animações lentas (300-500ms)
- Sem feedback abrupto; tudo é gradual e confortável

### Animation
- Fade-in ao entrar na viewport (duração: 600ms)
- Transições suaves entre estados (ease-in-out)
- Micro-animações em botões (escala 1.02, sem saltos)
- Scroll reveal: elementos aparecem de baixo para cima lentamente

### Typography System
- **Display**: Poppins Bold (28-36px) para títulos principais
- **Body**: Inter Regular (16px) para conteúdo
- **Accent**: Poppins SemiBold (18px) para destaques
- Hierarquia clara com espaçamento vertical generoso

## Abordagem 2: Warmth Emocional com Ilustração Humanizada
**Probabilidade:** 0.07

### Design Movement
Design humanista com influência de ilustração editorial e psicologia das cores quentes.

### Core Principles
1. **Acolhimento Visual**: Cores quentes que transmitem abraço e segurança
2. **Ilustração Narrativa**: Personagens e cenas que contam histórias de recuperação
3. **Comunidade Implícita**: Elementos que sugerem conexão e apoio mútuo
4. **Progressão Visual**: Jornada clara do luto para esperança

### Color Philosophy
- **Paleta Principal**: Coral suave, pêssego, dourado quente, com azul-índigo para contraste
- **Intenção Emocional**: Coral representa calor e compaixão; dourado evoca valor e renovação
- **Degradês Suaves**: Transições entre cores quentes criam movimento visual

### Layout Paradigm
- Alternância entre seções de conteúdo e ilustrações grandes
- Ilustrações ocupam 40-50% da tela em desktop
- Texto envolvido ao redor de elementos visuais (não centralizado)
- Uso de cards com sombra suave e gradiente de fundo

### Signature Elements
1. **Ilustrações Customizadas**: Personagens em diferentes estados emocionais
2. **Padrões Decorativos**: Flores, folhas, elementos naturais sutis
3. **Ícones Ilustrados**: Estilo consistente com personagens

### Interaction Philosophy
- Hover effects que revelam detalhes nas ilustrações
- Cliques produzem animações que reforçam emoção positiva
- Feedback visual que celebra ações do usuário

### Animation
- Entrance: ilustrações deslizam de um lado com fade (400ms)
- Hover: elementos ilustrados ganham pequeno bounce (200ms)
- Scroll: ilustrações parallax leve (velocidade reduzida)
- Transições entre telas com fade suave (300ms)

### Typography System
- **Display**: Playfair Display Bold (32-40px) para títulos (elegância)
- **Body**: Lato Regular (16px) para conteúdo (legibilidade)
- **Accent**: Playfair Display SemiBold (20px) para destaques
- Espaçamento gerado para respirabilidade

## Abordagem 3: Modernismo Dinâmico com Movimento Fluido
**Probabilidade:** 0.06

### Design Movement
Design moderno com influência de motion graphics e interfaces de aplicativos premium.

### Core Principles
1. **Movimento Contínuo**: Animações que criam sensação de fluxo e progresso
2. **Contraste Ousado**: Cores vibrantes contra fundos escuros ou claros
3. **Tipografia Expressiva**: Fontes grandes e impactantes
4. **Feedback Imediato**: Cada ação produz resposta visual clara

### Color Philosophy
- **Paleta Principal**: Roxo profundo, magenta, ciano, com branco para contraste
- **Intenção Emocional**: Roxo representa transformação; magenta é energia; ciano é clareza
- **Gradientes Dinâmicos**: Transições vibrantes entre cores

### Layout Paradigm
- Grid assimétrico com blocos de tamanho variável
- Elementos que se sobrepõem e criam profundidade
- Seções full-width com conteúdo fluindo em diferentes direções
- Uso de clip-path para criar formas geométricas ousadas

### Signature Elements
1. **Formas Geométricas Ousadas**: Triângulos, círculos que se intersectam
2. **Gradientes Animados**: Fundos que mudam cor ao scroll
3. **Tipografia Sobreposta**: Texto em camadas para profundidade

### Interaction Philosophy
- Hover effects que transformam elementos (rotação, escala)
- Cliques produzem explosões de cor ou partículas
- Feedback que celebra a ação de forma visual e energética

### Animation
- Entrance: elementos explodem na tela com spring physics (500ms)
- Hover: rotação suave (180 graus) com mudança de cor (300ms)
- Scroll: elementos seguem movimento do mouse (parallax agressivo)
- Transições entre telas com wipe ou slide dinâmico (400ms)

### Typography System
- **Display**: Montserrat Bold (36-48px) para títulos (impacto)
- **Body**: Roboto Regular (16px) para conteúdo (clareza)
- **Accent**: Montserrat ExtraBold (24px) para destaques
- Espaçamento compacto mas respirável

---

## Seleção Final

**Abordagem Escolhida: Minimalismo Terapêutico com Geometria Suave**

Esta abordagem foi selecionada porque:
- Alinha-se perfeitamente com o propósito terapêutico do aplicativo
- Cria um ambiente visualmente seguro e acolhedor
- Permite que o conteúdo emocional seja o foco, não a estética
- Facilita a legibilidade e acessibilidade
- Transmite profissionalismo e cuidado

### Decisões de Design Confirmadas
- **Cores**: Bege (#F5F1ED), Creme (#FFFBF7), Verde-água (#A8D5D0), Cinza escuro (#2C2C2C)
- **Tipografia**: Poppins para títulos, Inter para corpo
- **Espaçamento**: Generoso, com mínimo de 2rem entre seções
- **Animações**: Lentas e suaves, durações entre 300-600ms
- **Formas**: Bordas arredondadas (radius: 16-24px), sem ângulos agressivos
